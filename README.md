# Проект React - Аналоговые часы
React-приложение для отображения текущего времени в виде аналоговых часов

Учебный проект по курсу веб-разработки. Реализованы аналоговые часы с плавным движением стрелок и минималистичным дизайном.

## Описание
Приложение отображает аналоговые часы со следующими элементами:
- Часовая стрелка
- Минутная стрелка  
- Секундная стрелка
- Центральный циферблат

Все вычисления времени производятся на клиенте — бэкенд и интернет не требуются.

## Основные возможности
- Отображение реального времени
- Плавное движение стрелок
- Автоматическое обновление каждую секунду
- Минималистичный и чистый дизайн

## Математические расчеты
- Секундная стрелка: 6° в секунду (360° / 60)
- Минутная стрелка: 6° в минуту (360° / 60)  
- Часовая стрелка: 30° в час (360° / 12) + 0.5° в минуту

## Запуск локально

### Способ 1: Один HTML файл
```bash
# Скачайте index.html и откройте в браузере
open index.html
```

### Способ 2: React проект
```bash
# Клонируйте репозиторий:
git clone [ссылка-на-репозиторий]
cd analog-clock

# Установите зависимости:
npm install

# Запустите приложение:
npm start
```

## Особенности реализации
- Использование React хуков для управления состоянием
- CSS-анимации для плавного движения стрелок
- Оптимизированная перерисовка компонентов
- Кросс-браузерная совместимость
- Минимальный bundle размер
